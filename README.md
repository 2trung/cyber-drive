# Cyber Drive

An infinite synthwave driving scene built with React Three Fiber on WebGPU. Cruise down an endless neon highway, steer with your mouse, and hold left click to engage hyper-drive turbo.

![Cyber Drive](public/screenshot.png)

## Run

```
yarn
yarn dev
```

## Build

```
yarn build
```

## Deploy

```
yarn deploy
```

Publishes the production build to the `gh-pages` branch.
